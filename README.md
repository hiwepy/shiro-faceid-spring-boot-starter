# spring-boot-starter-shiro-faceid

### 说明


 > Shiro 整合人脸识别技术，实现  ukey + 人脸扫描 登录


### Maven

``` xml
<dependency>
	<groupId>${project.groupId}</groupId>
	<artifactId>spring-boot-starter-shiro-faceid</artifactId>
	<version>${project.version}</version>
</dependency>
```
